## 📱 Complete Feature Matrix

### CANVAS & DRAWING (The Core Product)

| Feature | FREE | PRO (₹499/yr) | FOUNDING (₹1,999) | Why? |
|:---|:---|:---|:---|:---|
| Infinite Canvas (pan, zoom, draw) | ✅ Full | ✅ Full | ✅ Full | This is the HOOK. If you limit this, nobody tries the app. |
| Pen Tool | ✅ | ✅ | ✅ | Basic drawing must be free |
| Pencil Tool | ✅ | ✅ | ✅ | Basic drawing must be free |
| Eraser | ✅ | ✅ | ✅ | Basic tool |
| Highlighter | ❌ | ✅ | ✅ | Upgrade nudge — students NEED this |
| Text Tool | ✅ | ✅ | ✅ | Basic tool |
| Shape Tool (lines, circles, rectangles) | ✅ Basic (3 shapes) | ✅ All shapes | ✅ All shapes | Free gets basics, Pro gets arrows, stars, etc. |
| Lasso Select & Move | ✅ | ✅ | ✅ | Basic editing |
| Pressure Sensitivity | ✅ | ✅ | ✅ | Hardware feature, don't gate it |
| Custom Brush Thickness | 3 presets | ✅ Full slider | ✅ Full slider | Soft nudge |
| Color Palette | 12 basic colors | ✅ Full color wheel + custom hex | ✅ Full color wheel + custom hex | This is a PROVEN upgrade trigger |
| Page Rotation | ✅ | ✅ | ✅ | Core differentiator, must be free |
| Dual Page Opening (Split View) | ✅ | ✅ | ✅ | Key productivity feature, should be free to hook users |

**Strategy:** The canvas itself is always fully infinite and free. We gate the **tools** (highlighter, full colors, brush sizes) — not the canvas. This way free users fall in love with the space but want better tools.

### 📁 ORGANIZATION (Folders & Notebooks)

| Feature | FREE | PRO (₹499/yr) | FOUNDING (₹1,999) | Why? |
|:---|:---|:---|:---|:---|
| Notebooks | 3 max | ✅ Unlimited | ✅ Unlimited | 3 is enough to try, not enough for a semester |
| Pages per Notebook | 10 max | ✅ Unlimited | ✅ Unlimited | Students will hit 10 pages in week 1 |
| Folder Nesting | 2 levels deep | ✅ Unlimited depth | ✅ Unlimited depth | Professionals need deep folders |
| Custom Notebook Covers | ❌ | ✅ | ✅ | Aesthetic upgrade, students love this |
| Notebook Sorting (by name, date, custom) | By date only | ✅ All options | ✅ All options | Soft nudge |
| Tags | ❌ | ✅ | ✅ | Organization power feature |
| Search (by notebook title) | ✅ | ✅ | ✅ | Basic feature |
| Search (inside handwriting - OCR) | ❌ | ✅ | ✅ | AI-powered, expensive to run, premium feature |

**Strategy:** 3 notebooks + 10 pages = enough for a student to try for 1-2 weeks. Then they hit the wall and upgrade. The folder nesting limit (2 levels) specifically targets professionals who need deeper organization.

### ☁️ SYNC & DEVICES

| Feature | FREE | PRO (₹499/yr) | FOUNDING (₹1,999) | Why? |
|:---|:---|:---|:---|:---|
| Cloud Backup | ✅ (to 1 device) | ✅ All devices | ✅ All devices | Sync is THE reason people pay |
| Real-time Sync | ❌ | ✅ | ✅ | Premium feature |
| Number of Devices | 1 | ✅ Unlimited | ✅ Unlimited | Our biggest moat vs Samsung Notes |
| Offline Mode | ✅ | ✅ | ✅ | Must work offline, basic expectation |
| Sync History / Version Control | ❌ | ✅ (30 days) | ✅ (Unlimited) | Founding gets extra value |

**Strategy:** Single-device cloud backup is free. But the moment a user gets a second device (phone + tablet, or tablet + laptop), they MUST upgrade. This is the #1 conversion trigger in the note app market.

## 🎯 The Top 5 Upgrade Triggers (In Order of Conversion Power)

These are the moments a free user thinks "OK, I need to pay":

1. **"I need more notebooks"** → They hit the 3 notebook limit
2. **"I need it on my laptop too"** → They want multi-device sync
3. **"I can't submit this with a watermark"** → They need clean PDF export
4. **"I want dark mode"** → Late night studying
5. **"I need the highlighter"** → Annotating PDFs/textbooks
