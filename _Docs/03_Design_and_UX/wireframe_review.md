## 📝 Feedback Guide

When reviewing, consider these questions for each screen:

| Question | What to Look For |
|:---|:---|
| **Layout** | Does the information hierarchy make sense? Are the most important elements prominent? |
| **Navigation** | Can you mentally trace the user flows? (Open app → Create notebook → Draw on canvas) |
| **Canvas UX** | Is the toolbar placement intuitive? Is the rotation button easy to find? |
| **Consistency** | Do all screens feel like they belong to the same app? |
| **Missing Elements** | Is anything missing that you expected to see? |

> [!TIP]
> To request changes, just tell me something like:
> - *"Make the onboarding screen 1 more minimalist"*
> - *"The paywall needs bigger pricing cards"*
> - *"Add a dark mode toggle to the settings screen"*
> 
> I'll use Stitch's `edit_screens` tool to update them instantly.
